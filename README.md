# ✅ Prince's Terminal To-Do App

A clean, emoji-driven and beginner-friendly Python CLI project that helps you manage your daily grind like a dev boss 👑  
No advanced libraries. Just vibes, files, and functionality 🔥

---

## ✨ Features

- 🔲 Add unlimited tasks  
- ✅ Mark tasks as completed  
- 🗑️ Delete tasks with ease  
- 💾 Auto-saves to `tasks.txt` locally  
- ⌛ `time.sleep()` for chill UX flow  
- 🧠 Fully beginner-readable & comment-rich code  
- 🤌 Aesthetic CLI menu with emoji style

---

## 🚀 How to Run

```bash
python todo_prince.py
```

> 💡 Make sure Python 3.x is installed and you're in the same directory.
